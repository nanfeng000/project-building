# Boundary Head 消融

> 在 full v2-lite (C) 主干上，对比是否启用轻量 D1 boundary aux head。
> 3 seeds = 42 / 123 / 3407，其它训练设置与主结果一致。
> Boundary head：weight=0.5，kernel=3，BCE + Dice boundary aux loss，仅在训练时激活。

**WHU 结论**：
- 逐 seed ΔIoU 全为正；ΔIoU mean = +0.0062 > max model std (0.0017) → **稳定提升**。
- Δboundary-IoU mean = +0.0198，所有 seed 上提升幅度均超过自身 std → **稳定提升边界质量**。
- 参数仅 +83k，速度几乎无变化 (~5.42 ms/img)。

## 跨数据集一致性

| Dataset | mean ΔIoU | mean Δboundary-IoU | 逐 seed ΔIoU 均为正 | Δ 是否超过 max std |
| ------- | --------- | ------------------ | ------------------- | ------------------ |
| WHU     | +0.0062   | +0.0198            | True                | True               |
| Inria   | +0.0071   | +0.0074            | True                | True               |

**总判定**：boundary head 在两个数据集上均带来**稳定且超 seed 波动的 IoU 与 boundary-IoU 收益**，参数与速度代价极小，是论文 ablation 中最干净的一组结论。
